# Polymarket vs FiveThirtyEight: A 2024 Election Polling Retrospective

A NeurIPS-format workshop paper comparing daily Polymarket prices against
FiveThirtyEight polling averages on 14 markets (13 states + national) over
the final ~3 months of the 2024 U.S. presidential election cycle.

## Reproducing the numbers in the paper

```bash
cd scripts
pip install -r requirements.txt
python fetch_data.py    # ~30s; pulls Polymarket Gamma/CLOB + 538 Wayback CSV
python analysis.py      # writes results.json (all paper numbers come from here)
python make_figures.py  # writes figures/*.png
```

Every number in `main.tex` is derivable from `results.json`; every figure
is regenerated by `make_figures.py`. The full pipeline runs in under a
minute on a laptop and requires no API keys.

## Files

- `main.tex` — paper source (NeurIPS 2024 preprint style)
- `references.bib` — 16 verified references (cross-checked via Semantic
  Scholar / publisher pages before inclusion)
- `neurips_2024.sty` — NeurIPS official style file
- `scripts/fetch_data.py` — pulls Polymarket markets + CLOB price history
  for 14 markets, plus the 5 Nov 2024 Wayback snapshot of 538's
  `presidential_general_averages.csv`
- `scripts/analysis.py` — produces `results.json` (Brier scores, Pearson
  correlations, sign and Wilcoxon tests, lead/lag, calibration buckets)
- `scripts/make_figures.py` — produces all four figures
- `scripts/requirements.txt` — pinned dependency versions
- `data/` — raw inputs (regenerated by `fetch_data.py`; ~2MB)
- `figures/` — final paper figures (~450KB)
- `results.json` — every numeric result quoted in the paper
